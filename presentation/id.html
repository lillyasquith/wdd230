ID Selectors

An ID selector begins with a # character, it is used to select one specific element. Therefore, an ID selector is used to style one single HTML element. ID elements can only have a unique id value applied to each of them. In other words, each HTML ID element must have separate names/values.

Example

This CSS rule below will be applied only to the HTML element with id="video":

#video {

width: 100%;

}
