Difference Between ID vs. Class Selectors

Class ID
  • Select a group of elements
  • Is written with a (.) followed by the class name
  • Select multiple elements with that class.
  • Select a single element
  • Is written with a (#) symbol followed by the ID name
  • Select a single element that id.
