Types of Selectors

There are a few different groupings of selectors, we can put them into five categories:

  1. Simple selectors
  2. Combinator selectors
  3. Pseudo-class selectors
  4. Pseudo-elements selectors
  5. Attribute selectors

Simple selectors

The elements are selected based on name, id, or class.

Example

Select the (h1) element

h1 {

}

Select the element with id="video"

#video {

}

Select the element with class="nav"

.nav {

}

Select all HTML element on the page (universal/global selector).

* {

}

Select all (div) elements and all (p) elements

div, p {

}

Select only (p) elements with class="intro"

p.intro {

}

Combinator selectors

The elements are selected based on a specific relationship between them.

There are four different combinators in CSS:

  • Descendant selector (space)
  • Child selector (>)
  • Adjacent sibling selector (+)
  • General sibling selector (~)

Example

Selects all (p) elements where the parent is a (div) element

div > p {

}

Pseudo-class selectors

The elements are selected based on a certain state.

We can use Pseudo-class selectors to:

  • Style an element when a user mouses over it.
  • Style visited and unvisited links differently.
  • Style an element when it gets focus.

Example

Use the :hover pseudo-class on a (div) element

div:hover {

}

Pseudo-elements selectors

We select and style a part of an element.

It can be used to:

  • Style the first letter, or line of an element.
  • Insert content before, or after the content of an element.

Example

The ::first-line pseudo-element is used to add a special style to the first line of a text in all (p) elements.

p::first-line {

}

Attribute selectors

The elements are selected based on an attribute or attribute value.

Example

This example selects all (a) elements with a target attribute

a[target] {

}
